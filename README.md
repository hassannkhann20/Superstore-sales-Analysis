Superstore Sales Analysis (Python + Power BI)

Project Overview
This project is an end-to-end data analytics solution built using Python and Power BI.
The objective is to analyze sales performance, uncover key business insights, and
present them through a clean, interactive Power BI dashboard.

The project follows a complete analytics workflow:
data cleaning → exploratory data analysis (EDA) → visualization → business insights.

---

  Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI
- GitHub
